STEP 1: REQUIREMENTS INSTALL KARO
----------------------------------
- Python 3.6+ installed hona chahiye
- requirements.txt file ke saath ye command run karo:
  pip install -r requirements.txt

STEP 2: BACKGROUND CAPTURE KARO
--------------------------------
1. background.py run karo:
   python background.py
2. Camera khulegi, apna background set karo
3. 'Q' press karke image save karo

STEP 3: INVISIBILITY CLOAK CHALAO
----------------------------------
1. Red cloth leke camera ke saamne khade ho jao
2. invisible_cloak.py run karo:
   python invisible_cloak.py
3. Ab red cloth invisible dikhega!

AUTOMATIC RUN:
-------------
Windows: run.bat file double-click karo
Mac/Linux: terminal mein ./run.sh type karo

TIPS:
-----
- Lighting acchi honi chahiye
- Plain red cloth use karo
- Background static hona chahiye
